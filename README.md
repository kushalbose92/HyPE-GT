# HyPE-GT: where Graph Transformers meet Hyperbolic Positional Encodings

## Dependencies

* Pytorch 1.9.0
* CUDA 11.1
* Numpy 1.22.3
* DGL 0.9.1post1
* OGB 1.3.6

## Usage
 To run the codes for ZINC, PATTERN, and CLUSTER
 ```
 cd hyper-gt-framework
 
 sh run_zinc.sh
 sh run_sbms.sh
 ```
 
 To run the codes for ogbg-molhiv
 ```
 cd ogbg-molhiv
 
 sh run_molhiv.sh
 '''
